This directory contains publication-quality figures generated by `../generate_figures.py`.

| `Figure_1_Dimensions.mmd` | Four Dimensions mindmap | Mermaid |

## How to Render

### matplotlib figures (Python)
```bash
cd /Users/ssanjay/GitHub/interactive_map/theory
python generate_figures.py
```

### Mermaid figures
Option 1: Use VS Code extension "Markdown Preview Mermaid Support"
Option 2: Use [mermaid.live](https://mermaid.live) online editor
Option 3: Use mermaid-cli (`mmdc -i file.mmd -o file.svg`)

```bash
# Install mermaid-cli
npm install -g @mermaid-js/mermaid-cli

# Render to SVG
mmdc -i Figure_1_Dimensions.mmd -o Figure_1_Dimensions.svg
mmdc -i Figure_2_DecisionTree.mmd -o Figure_2_DecisionTree.svg
```
